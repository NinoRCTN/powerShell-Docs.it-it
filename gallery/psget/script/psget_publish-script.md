# Publish-Script

Il cmdlet Publish-Script pubblica lo script specificato nella raccolta in linea.

<a id="description" class="xliff"></a>
## Descrizione

Il cmdlet Publish-Script consente di pubblicare il file di script con metadati validi come Version, Guid, Author, Description e cos√¨ via. Il parametro opzionale Force nel cmdlet Publish-Script avvia il file NuGet.exe senza chiedere conferma.

<a id="cmdlet-syntax" class="xliff"></a>
## Sintassi del cmdlet

```powershell
Get-Command -Name Publish-Script -Module PowerShellGet -Syntax
```
