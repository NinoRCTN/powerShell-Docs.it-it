# <a name="stopproc01-code-samples"></a>Codici di esempio di StopProc01

Di seguito è riportato l'esempio di codice per il cmdlet di esempio StopProc01. Questo è il `Stop-Process` cmdlet esempio descritto in [creazione di un Cmdlet che modifichi il sistema](../cmdlet/creating-a-cmdlet-that-modifies-the-system.md). Il `Stop-Process` cmdlet è progettato per arrestare i processi che vengono recuperati usando il cmdlet Get-Proc (descritto in [creazione del primo Cmdlet](../cmdlet/creating-a-cmdlet-without-parameters.md)).

> [!NOTE]
> È possibile scaricare il C# file di origine (stopproc01.cs) per il cmdlet Stop-Process tramite il Microsoft Windows Software Development Kit per Windows Vista e i componenti di Runtime di .NET Framework 3.0. Per istruzioni sul download, vedere [come installare Windows PowerShell e il Download di Windows PowerShell SDK](/powershell/developer/installing-the-windows-powershell-sdk).
>
> Sono disponibili in file di origine scaricato il  **\<esempi di PowerShell >** directory.
