# <a name="cmdlet-development-guidelines"></a>Linee guida sullo sviluppo relative ai cmdlet

Negli argomenti di questa sezione forniscono le linee guida di sviluppo che è possibile usare per produrre i cmdlet in formato corretto. Sfruttando la funzionalità comune fornita dal runtime di Windows PowerShell e attenendosi alle linee guida, è possibile lo sviluppo di cmdlet affidabili con il minimo sforzo e fornire all'utente un'esperienza coerente. Inoltre, si ridurrà il carico di test perché la funzionalità comune non richiedono la riesecuzione test.
