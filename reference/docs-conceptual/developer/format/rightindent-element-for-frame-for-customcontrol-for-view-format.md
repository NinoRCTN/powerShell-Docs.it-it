# <a name="rightindent-element-for-frame-for-customcontrol-for-view-format"></a>Elemento RightIndent per Frame per CustomControl per View (formato)

Specifica il numero di caratteri spostati al di fuori del margine destro da parte dei dati. Questo elemento viene utilizzato quando si definisce una visualizzazione controlli personalizzata.
