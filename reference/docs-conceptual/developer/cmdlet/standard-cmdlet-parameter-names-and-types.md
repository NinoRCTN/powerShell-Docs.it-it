# <a name="standard-cmdlet-parameter-names-and-types"></a>Tipi e nomi standard per i parametri dei cmdlet

I nomi dei parametri del cmdlet devono essere coerenti tra i cmdlet progettati. Negli argomenti seguenti vengono elencati i nomi dei parametri che si consiglia di utilizzare quando si dichiarano i parametri del cmdlet. Gli argomenti descrivono inoltre il tipo di dati e la funzionalità consigliati di ogni parametro.
