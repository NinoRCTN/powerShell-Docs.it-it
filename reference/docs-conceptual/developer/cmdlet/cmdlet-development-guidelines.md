# <a name="cmdlet-development-guidelines"></a>Linee guida sullo sviluppo relative ai cmdlet

Negli argomenti di questa sezione vengono fornite linee guida per lo sviluppo che è possibile utilizzare per produrre cmdlet ben formati. Sfruttando le funzionalità comuni fornite dal runtime di Windows PowerShell e seguendo queste linee guida, è possibile sviluppare cmdlet affidabili con il minimo sforzo e fornire all'utente un'esperienza coerente. Inoltre, sarà possibile ridurre il carico di test perché la funzionalità comune non richiede la riprova.
