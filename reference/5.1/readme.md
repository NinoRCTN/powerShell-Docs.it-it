# <a name="windows-powershell-51"></a>Windows PowerShell 5.1

Ultimo aggiornamento: ottobre 2016
- Si applica a: Windows PowerShell 2.0, Windows PowerShell 3.0, Windows PowerShell 4.0, Windows PowerShell 5.0, Windows PowerShell 5.1

Questa sezione contiene gli argomenti della Guida relativi ai cmdlet, quelli relativi ai provider, nonché gli argomenti concettuali ("about_") riguardanti i moduli installati per impostazione predefinita con Windows PowerShell versione 5.1.

La tabella seguente illustra la versione pubblicata più recente della Guida per ogni modulo.
La Guida è disponibile tramite i collegamenti seguenti o come file della Guida aggiornabile.
La Guida aggiornabile, introdotta con Windows PowerShell 3.0, consente di avere a disposizione gli argomenti della Guida più recenti in locale nel computer.
Per altre informazioni, vedere about_Updatable_Help.
